# Call-center-Soporte-Analistas-local-tool-TG-CW
Herramienta local desarrollada en HTML y JavaScript para apoyar las labores diarias de soporte técnico en entornos de call center.  Este proyecto está diseñado para uso **100% local**, sin instalaciones adicionales, y enfocado en mejorar la eficiencia operativa (AHT) durante la atención de incidentes.
